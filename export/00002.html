온라인에서 원하는 문서 서식을 내려받을 수 있습니다. 다양한 디자인이 적용된 문서를 통해 간편하고 빠르게 자료를 정리하거나 문서를 생성할 수 있습니다.

 따라 하기

문서 서식 이용하기

  • 1.[도구] 도구 상자에서 [문서 서식 내려받기]를 누릅니다.

  • 2.[문서 서식 내려받기] 대화 상자의 왼쪽 영역에서 원하는 서식 범주를 선택합니다.
    선택한 서식 범주에 속하는 다양한 문서가 대화 상자의 오른쪽 영역에 미리 보기 형태로 표시됩니다.

  • 3.대화 상자의 오른쪽 영역에서 원하는 디자인의 문서를 선택합니다.

  • 4.선택한 문서의 미리 보기 창에서 [다운로드] 단추를 누릅니다.

  • 5.선택한 문서가 프로그램 문서 창에 열리면 문서를 편집한 다음 저장합니다.

 자세한 정보

[문서 서식 내려받기] 대화 상자

문서 서식

문서 특성별로 분류하여 사용자가 사용 목적에 맞춰 문서를 쉽게 찾고 선택할 수 있습니다.

검색 입력 상자

사용 목적과 일치하는 문서만 선별적으로 검색합니다.
일반적으로 사용하는 문서 위주로 문서 서식을 제공하고 있어서 입력 내용에 따라 검색 결과를 얻지 못할 수도 있습니다.

미리 보기 목록

문서 제목과 문서에 적용된 서식을 축소판 형태로 표시합니다.

 참고

문서 불러오기와 저장하기

[문서 서식 내려받기] 대화 상자에서 문서를 다운로드하면 해당 문서를 편집 창에 불러오는 것으로 내 컴퓨터에 저장하는 것은 아닙니다. 다운로드한 문서를 저장하면 최근 문서 목록에서 빠르게 불러올 수 있으며 해당 문서를 다시 다운로드할 필요 없이 언제든지 이용할 수 있습니다.

문서 저작권

(주)한글과컴퓨터는 한컴오피스 사용자가 간편하고 빠르게 자료를 정리하거나 문서를 생성할 수 있도록 다양한 서식의 문서를 제공합니다. [문서 서식 내려받기]를 통해 다운로드한 문서는 저작권이 보호되는 문서이므로 무단으로 배포하거나 판매할 수 없습니다.
